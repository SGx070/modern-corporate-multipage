# Nexus Digital | Professional Agency Suite

**Nexus Digital** is a high-performance, 10-page multi-page website built to simulate a modern technology agency. This project focuses on clean technical architecture, a sophisticated "Deep Space" dark theme, and a seamless user experience across all devices.

## 🚀 Live Demo
https://sgx070.github.io/Nexus-digital/

---

## 🛠️ Tech Stack
* **HTML5:** Semantic architecture for SEO and accessibility.
* **CSS3:** Custom variables, Flexbox, and CSS Grid for a fully responsive layout.
* **JavaScript (Vanilla):** Interactive components including FAQ accordions and form handling.
* **Design:** Modern Dark Theme with Glassmorphism effects.

---

## 📂 Project Structure (10-Page Architecture)
This repository contains a complete business website flow:
1. **Home:** High-impact landing page with Call-to-Action sections.
2. **About:** Company mission and team background.
3. **Services:** Detailed breakdown of agency solutions.
4. **Portfolio:** Grid-based showcase of featured projects.
5. **Reviews:** Client testimonials and trust-building social proof.
6. **FAQ:** Interactive accordion for common user inquiries.
7. **Contact:** Fully designed professional lead generation form.
8. **Careers:** Modern job listing page for company growth.
9. **Privacy Policy:** Standard legal documentation for data transparency.
10. **404 Page:** Custom error handling for improved User Experience (UX).

---

## ✨ Key Features
* **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop using Media Queries.
* **Modern UI:** Features a translucent, blurred navigation bar and neon-glow hover states.
* **Technical Optimization:** Corrects "Quirks Mode" and "CORB" issues for high-speed delivery.
* **User Navigation:** Intuitive footer and "Explore More" blocks to prevent dead-end pages.
